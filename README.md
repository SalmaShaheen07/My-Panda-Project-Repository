# 📊 Python & Pandas Data Analysis Project — Data Technician Bootcamp (2026)
This project demonstrates the core data analysis skills I developed during the Data Technician Bootcamp (May–June 2026), with a focus on using Python and Pandas to explore, clean, and analyse retail and sales datasets. The work was completed through a series of Google Colab notebooks and dataset‑based exercises, as described in the workbook:

“Go through the notebook and complete the exercises at the end… Download the dataset… Completed.”

## 🔧 Key Skills Demonstrated
### 1. Data Loading & Inspection
Using Pandas to import CSV files and inspect data structure, including:

df.head(), df.info(), df.describe()

Checking column types and identifying missing values
These steps supported the initial exploration of retail and sales datasets referenced in the workbook tasks.

“Download the dataset… Go through the notebook and complete the exercises…”

### 2. Filtering & Subsetting Data
I used Pandas indexing tools to extract meaningful subsets of data:

.loc[] for label‑based filtering

.iloc[] for position‑based selection

Conditional filtering (e.g., selecting high‑value customers or specific product categories)

These techniques allowed me to isolate patterns and trends within the sales data.

### 3. Data Cleaning & Preparation
I applied essential data‑cleaning methods to prepare datasets for analysis:

Handling missing values with dropna() and fillna()

Removing duplicates

Renaming and reformatting columns

Converting data types for accurate calculations

This reflects the workbook’s emphasis on completing dataset‑based exercises and preparing data for analysis.

“Go through the notebook and complete the exercises at the end.”

### 4. Data Manipulation with Pandas
I used a range of Pandas functions to explore and summarise retail and sales trends:

groupby() for aggregating sales by region, product, or customer type

sort_values() to rank products or categories

value_counts() for frequency analysis

agg() for multi‑metric summaries

These methods helped uncover insights such as top‑selling products, customer behaviour patterns, and seasonal trends.

### 5. Data Visualisation
To communicate insights effectively, I created visualisations using:

Pandas built‑in .plot()

Matplotlib for bar charts, line charts, and distribution plots

This aligns with the workbook’s recommended resources on data visualisation and Pandas tutorials:

“Kaggle Data Visualization… Kaggle Pandas Tutorial… Learn Pandas in Under 3 Hours | Filtering, Joins, Indexing, Data Cleaning, Visualizations.”
